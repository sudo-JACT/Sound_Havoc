Sound Havoc Amplifier

About the Plugin

This plugin changes the amplitude of the input audio signal. It uses a simple amplification algorithm that increases or decreases the signal level according to the specified parameters. The user can adjust the gain (gain) parameter to increase or decrease the amplitude of the input signal. This plugin can be used to increase the volume of an audio signal without unwanted distortion.
